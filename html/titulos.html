Os tipos de elementos: <h1>

Esse elemento e utilizado para colocar diversos tamanhos de letras, o <h1> sendo utilizado para o titulo

Esse elemento tem 6 variações indo do <h1> até o <h6>, o <h1> sendo maior e o <h6> o menor

Um exemplo desse elemento sendo utilizado sao as frase anteriore, a primeira sendo <h1> para representar o titulo
